# Frontend Mentor - Challenge #1

Esta es mi solución para el desafío **[Stats preview card component]** de Frontend Mentor. El objetivo de este proyecto fue practicar habilidades de desarrollo frontend mediante la implementación de un diseño proporcionado por la plataforma.

## 📋 Descripción

Este proyecto forma parte de los retos de Frontend Mentor y fue desarrollado siguiendo las especificaciones del diseño original.

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Responsive Design

## 🎯 Objetivos del reto

* Reproducir el diseño proporcionado.
* Aplicar buenas prácticas de maquetación.
* Mejorar habilidades de CSS y JavaScript.
* Desarrollar una interfaz adaptable a diferentes dispositivos.

## 📂 Cómo ejecutar el proyecto

Clona el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Accede al directorio:

```bash
cd nombre-del-proyecto
```

Abre el archivo `index.html` en tu navegador.
